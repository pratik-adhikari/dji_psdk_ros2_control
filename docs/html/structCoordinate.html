Detailed Description

Represents a 3D coordinate. The meaning of longitude/latitude fields depends on the transformation:

A simple struct holding x, y, z coordinates, used for local transformations.

  • For WGS84/EPSG:4326, 'latitude' is lat (deg), 'longitude' is lon (deg).
  • For UTM frames, 'longitude' = Easting (m), 'latitude' = Northing (m).
