Detailed Description

Converts GPS coordinates (WGS84) to a target frame (UTM or LOCAL) and vice versa.

For EPSG-based transforms (e.g., EPSG:4326 -> EPSG:32615), PROJ expects geographic input as (latitude, longitude).

This code will carefully swap as needed if the user passes (longitude, latitude) in the input struct, to maintain a user-friendly API.
